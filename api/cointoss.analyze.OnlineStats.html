OnlineStats

Utility to incrementally calculate mean, variance and standard deviation of a sample. Sample points can be #add(double), #remove(double) or #replace(double, double).

The implementation is based on Welford’s Algorithm given in Knuth Vol 2, p 232.

This class is NOT thread safe.

Adds the value x to the sample. The sample count is incremented by one by this operation,

Removes the value x currently present in this sample. The sample count is decremented by one by this operation.

replace(doublex, doubley)void

doublex

the value to remove
doubley

the value to add

Replaces the value x currently present in this sample with the new value y. In a sliding window, x is the value that drops out and y is the new value entering the window. The sample count remains constant with this operation.

getMean()double

double

the mean value of the sample

Returns the mean value of the sample. Returns 0 if the sample count is zero.

The method returns the calculated value and returns immediately.

getVariance()double

double

the biased variance of the sample

Returns the variance of the sample using the (n) method. Returns NaN if count is 0.

The method is based on calculated values and returns almost immediately (involves a simple division).

getVarianceUnbiased()double

double

the variance of the sample (bias corrected)

Returns the variance of the sample using the (n-1) method. Returns 0 if the sample count is zero, and Inf or NaN if count is 1.

The method is based on calculated values and returns almost immediately (involves a simple division).

getStdDev()double

double

the biased standard deviation of the sample

Returns the standard deviation of the sample using the (n) method. Returns NaN if count is 0.

The method is based on calculated values and returns almost immediately (involves a square root and division operation).

getStdDevUnbiased()double

double

the standard deviation of the sample (bias corrected)

Returns the standard deviation of the sample using the (n-1) method. Returns 0 if the sample count is zero, and Inf or NaN if count is 1.

The method is based on calculated values and returns almost immediately (involves a square root and division operation).

reset()void

Resets this sampler to its initial state. The sample count is 0 after this operation.

combine(OnlineStatswith)void

OnlineStatswith

the sampler with which this sampler is combined

Combines this sampler with the specified other sampler. After the operation, this sampler reflects the combined mean, variance and standard deviation.

Combining samplers is sometimes useful e.g. if separate parts of a statistic are collected on separate threads. The combine operation (maybe after calling clone) can be used to calculate the combined statistics with a lower frequency (e.g. every 1000 data points).
